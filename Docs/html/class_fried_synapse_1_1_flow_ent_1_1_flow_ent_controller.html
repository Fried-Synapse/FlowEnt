Detailed Description

A singleton monobehaviour that is the core of the FlowEnt library. Hold the main update method and a handful of global parameters. The script automatically attaches itself to an object in the scene called FlowEnt(created by this script). Make sure not to delete this object because the library will not work without this script attached to an object in the scene.
