FlowEnt.BezierSpline.BezierSpline ( params Vector3[]  points)

Creates a Bezier Spline. This is a spline based on the Bezier Curve.

Parameters
pointsThe sequence of points that will create the spline.</param

If 2 points are sent p0 will be startPoint and startControl, and p1 will be endControl and endPoint. If 3 points are sent p0 will be startPoint, p1 will be startControl and endControl, and p2 will be endPoint. If 4 points are sent p0 will be startPoint, p1 will be startControl, p2 will be endControl, and p3 will be endPoint. If 5 or more points are sent it'll create a spline with a bezier movement between the points.
